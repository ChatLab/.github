# ChatLab

> "The words you've typed shouldn't disappear into the scroll."

ChatLab is an open-source project for importing, analyzing, and understanding
your chat history — across WhatsApp, LINE, Discord, Instagram, Telegram and more.

No uploads. No subscriptions. Your data stays on your device, always.

With ChatLab, you can:

- 🔍 Search and retrieve any message across all your chat history
- 🧠 Chat with your history using AI — ask anything, get instant answers
- 📊 Analyze relationships, activity patterns, and key moments
- 🔌 Connect to any AI platform via CLI — make your memory available everywhere
